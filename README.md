Big rent store

## install and development
To start develop, run:
```bash
yarn install
yarn dev-fast
```
## build
To start build, run:
yarn build

## start server
To start server upon a build, run:
yarn build
yarn start
```
url - localhost:3000 
```

and open browser with displayed link
